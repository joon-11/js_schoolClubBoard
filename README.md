###### 2024.8 (1人 개인 프로젝트)

## 📌 Summary

**동아리 소통 게시판**

- 동아리원들의 원활한 소통과 교류를 위한 게시판.

<br />

## 🤔 Background

동아리원들 끼리 코드공유, 소통을 통해 서로 배우고 가르치라고 만들었다.

회원가입 시 동아리 담당 선생님만이 사용할 수 있는 secret key 를 사용하여 이 key를 보유하지 않은 사람은 회원가입 할 수 없다.

이 key는 선생님이 원하는대로 바꿀수 있으며 선생님은 모든 게시물, 댓글 등을 관리할 수 있다.
